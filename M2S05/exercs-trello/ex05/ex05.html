O que pode ensinar sobre a diferença de BOM e DOM ?

DOM são os objetos que você conhece do HTML (links, ids, etc) já o BOM são os objetos do Navegador (History, Navigator, Location, até o próprio DOM é um objeto do Navegador)

O que são seletores ?

Definem os elementos ao qual serão aplicados os estilos no CSS para alterar a aparência dos elementos de uma página HTML

Qual é a principal diferença funcional entre getElementById e querySelector ?

  • Com o ( querySelector ) retorno o primeiro elemento especificado pelo seletor

  • Ja com o (getElementById ) o elemento com o (Id) especificado é retornado.

    • O que são eventos ? Cite alguns exemplos.

    Eventos são ações realizadas em um determinado elemento da página web.

    Exemplos:

  • onBlur : Remove o foco do elemento
  • onClick: O elemento é clicado pelo usuário
  • onLoad: Carrega o elemento por completo
